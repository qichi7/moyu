## 动画系统

### 滑动/移动动画

- [ ] CSS transition监听的属性与JavaScript修改的属性一致
- [ ] 动画时长合理（150-200ms）
- [ ] 动画缓动函数合适（cubic-bezier）
- [ ] 不移动的元素是否被排除动画
- [ ] 移动时有视觉增强效果（scale、shadow、trail）
- [ ] 坐标转换逻辑正确（上下/左右移动特殊处理）

### 常见问题

| 问题 | 症状 |
|------|------|
| 动画不生效 | 元素瞬间跳变，无过渡效果 |
| 动画路径不明显 | 用户难以追踪移动轨迹 |
| 不该动的元素有动画 | 位置不变的元素也有动画效果 |
| 动画错乱/洗牌效果 | 元素移动到错误位置 |

## 移动控制

### 键盘控制

- [ ] 方向键响应灵敏
- [ ] 禁止反向移动（吃豆人类游戏）
- [ ] 转向必须在格子中心（防止穿墙）
- [ ] 按键去抖处理

### 触屏控制

- [ ] 监听整个document而非特定容器
- [ ] 排除输入框、按钮、弹窗等元素
- [ ] 滑动时间限制宽松（800ms+）
- [ ] passive:false才能调用preventDefault
- [ ] 摇杆与键盘控制共存

### 常见问题

| 问题 | 症状 |
|------|------|
| 滑动区域受限 | 只能在特定区域滑动有效 |
| 方向键不灵敏 | 按键有时无响应 |
| 撞墙反向运动 | 碰到墙壁后反向移动 |
| 启动时方向错误 | 游戏开始时角色朝向错误方向 |

## 碰撞检测

### 基础检测

- [ ] 边界检测完整
- [ ] 墙壁碰撞准确
- [ ] 角色间碰撞正确
- [ ] 道具碰撞正确

### 高级检测

- [ ] 相向移动穿过问题
- [ ] 移动轨迹中间位置判定
- [ ] 碰撞时位置修正
- [ ] 出生位置不重叠

### 常见问题

| 问题 | 症状 |
|------|------|
| 穿墙bug | 角色穿过墙壁 |
| 相向穿过 | 两个角色相向移动时穿过彼此 |
| 碰撞检测不准确 | 偶尔穿过或不该碰撞时碰撞 |
| 出生位置重叠 | 多个角色/道具生成在同一位置 |

## 移动端适配

### 布局适配

- [ ] 宽度根据viewportWidth动态计算
- [ ] 考虑body padding + container padding + gap
- [ ] 字体大小动态调整
- [ ] 居中显示（margin: 0 auto）
- [ ] 不同屏幕尺寸测试（小屏/大屏）

### iOS Safari特殊处理

- [ ] overflow: hidden防止页面滚动
- [ ] touch-action: none禁止触摸操作
- [ ] 使用@supports检测iOS Safari
- [ ] position: fixed锁定页面

### 响应式处理

- [ ] resize事件防抖处理（100ms+）
- [ ] 移动端使用display: block
- [ ] 桌面端保持inline-block或flex

### 常见问题

| 问题 | 症状 |
|------|------|
| 宽度不自适应 | 小屏幕显示不完整 |
| 界面不居中 | 网格偏向一侧 |
| iOS滚动问题 | 滑动触发下拉刷新 |
| resize卡顿 | 频繁重绘导致性能问题 |

## 排行榜系统

### 数据存储

- [ ] Gist ID硬编码
- [ ] Token仅在保存时输入
- [ ] 数据格式验证
- [ ] 类型匹配（字符串vs数字）

### 缓存机制

- [ ] 缓存过期时间合理（30秒）
- [ ] 刷新时清除缓存
- [ ] 缓存版本控制

### UI交互

- [ ] 刷新时保持当前选择状态
- [ ] 标签切换正确
- [ ] 筛选条件类型一致
- [ ] 第一名特殊显示

### 安全问题

- [ ] XSS防护（用户输入转义）
- [ ] Token不暴露在代码中
- [ ] Token存储在sessionStorage

### 常见问题

| 问题 | 症状 |
|------|------|
| 读取失败 | 类型不匹配导致筛选失败 |
| 刷新状态丢失 | 刷新后跳回默认标签 |
| 显示错误 | 筛选条件与数据类型不一致 |
| XSS漏洞 | 用户输入未转义导致脚本注入 |
