# 🎮 Moyu Games - 摸鱼游戏合集

一个有趣的网页游戏合集项目。

---

## 游戏列表

### 🌅 GoodMorning - 多人社交游戏
- **简介**: 一个可爱的多人在线社交游戏，玩家可以在地图上自由移动、聊天互动
- **功能**: 角色创建、自定义外观、多人在线、实时聊天
- **链接**: [https://qichi7.github.io/moyu/game/goodmorning-game/](https://qichi7.github.io/moyu/game/goodmorning-game/)
- **目录**: [goodmorning-game/](goodmorning-game/)

### 🍜 中午吃点啥 - 午餐选择轮盘
- **简介**: 一个轮盘抽奖网页，帮助随机决定中午吃什么
- **功能**: 轮盘抽奖、添加选项、历史记录、自动排除最近三次吃过的
- **链接**: [https://qichi7.github.io/moyu/game/lunch-wheel/](https://qichi7.github.io/moyu/game/lunch-wheel/)
- **目录**: [lunch-wheel/](lunch-wheel/)
- **数据存储**: [Gist](https://gist.github.com/qichi7/8db89a5ec373b9e93642971d839a8e49)

## 技术栈

- HTML5 + CSS3 + JavaScript
- Canvas 绘图
- GitHub Gist 数据存储
- GitHub Pages 部署
