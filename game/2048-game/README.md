# 2048 游戏

经典2048游戏，支持自定义网格大小、动画效果、撤销功能和排行榜。

## 功能特性

- 🎮 经典2048玩法
- 📏 支持3×3到6×6网格大小
- 🎨 流畅的动画效果（移动、合并、新方块）
- ↩️ 撤销功能（Ctrl+Z，最多10步）
- 🏆 GitHub Gist排行榜
- 💾 本地存储（刷新页面不丢失进度）
- 🎉 游戏胜利提示（达成2048）
- 📱 移动端触摸支持
- ✨ 分数动画效果

## 如何玩

### 键盘控制
- ⬆️ 上箭头：向上移动
- ⬇️ 下箭头：向下移动
- ⬅️ 左箭头：向左移动
- ➡️ 右箭头：向右移动
- `Ctrl+Z` 或点击撤销按钮：撤销上一步操作

### 触摸控制（移动端）
- 向上滑动：向上移动
- 向下滑动：向下移动
- 向左滑动：向左移动
- 向右滑动：向右移动

## 游戏规则

1. 使用方向键移动所有方块
2. 相同数字的方块碰撞时会合并
3. 每次移动后会随机生成一个新方块（90%是2，10%是4）
4. 目标是合并出2048方块
5. 无法移动时游戏结束

## 新增功能

### 动画效果
- 方块移动时有平滑过渡
- 方块合并时有缩放动画
- 新方块出现时有弹跳效果
- 分数增加时有数字弹出和浮动提示

### 撤销功能
- 点击撤销按钮或使用 `Ctrl+Z` 可撤销上一步操作
- 最多可撤销10步
- 游戏结束后不可撤销

### 本地存储
- 游戏进度自动保存到本地存储
- 刷新页面或关闭浏览器后可恢复进度
- 每种网格大小独立保存

### 游戏胜利
- 达成2048方块时会弹出胜利提示
- 可以选择"继续挑战"继续游戏
- 也可以选择"新游戏"重新开始

## 网格大小

可选网格大小：
- **3×3** - 更快达成目标，难度较高
- **4×4** - 经典大小
- **5×5** - 更大空间，更容易
- **6×6** - 超大空间，最简单

## 排行榜

排行榜数据存储在GitHub Gist中。

⚠️ **注意**：排行榜保存功能仅对Gist所有者可用，其他玩家可以查看但无法保存。

### Token安全
- Token使用sessionStorage存储
- 关闭浏览器后Token自动失效
